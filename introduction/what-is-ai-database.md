# 什么是 AI Database

AI Database 是一款专为 AI 应用设计的全功能数据库，集成了数据分析、多模态检索（向量、全文、JSON、GIS）与 ACID 事务于一体，构建出统一的智能数据服务平台，可作为 AI 系统的后端核心。

AI Database 全面兼容 PostgreSQL 协议，开发者可以直接使用熟悉的 SQL 语法与工具无缝接入，无需数据迁移即可获得更强大的 AI 原生能力。基于 Serverless 架构，AI Database 能够按需扩缩容，在保证性能的同时实现极致弹性与成本优化，帮助企业以更低成本、更为高效率构建智能化应用与数据驱动的 AI 系统，加速实现智能分析、语义检索与生成式 AI 等场景的落地。

## 核心能力

### 多模态检索一体化​

- **向量检索**​：支持高维向量相似度计算，适配大模型嵌入向量快速匹配（如推荐系统、语义搜索）

- **全文检索**​：内置倒排索引，实现文本关键词的高效查询与模糊匹配
        
- **JSON检索**​：原生支持 JSON 文档存储与路径查询，灵活处理半结构化数据
        
- **GIS空间检索**​：集成 PostGIS 兼容接口，支持空间关系计算（如距离、包含）
        
### AI-Native 设计
    
- **ACID事务保障**​：确保 AI 训练数据与业务数据的强一致性，避免脏读/幻读问题
        
- **数据分析融合**​：支持SQL窗口函数、复杂聚合与实时计算，无缝衔接BI工具与AI模型
        

### Serverless架构
    
- **秒级启停**​：根据负载动态扩缩容，冷启动时间<1秒，适合间歇性访问的AI场景（如批处理任务）
