# 什么是 AI Database

AI Database 是面向新时代智能应用场景构建的数据基础设施，它将数据分析、知识检索与事务处理融合在一体，帮助企业在统一的数据底座上连接大模型与业务系统。相比于传统数据库只能存储结构化数据、矢量库只能做单一检索的割裂方案，AI Database 通过融合向量、全文、JSON、GIS 等多模态存储与查询能力，为 AI 应用提供更贴近业务语义的知识检索体验。同时，AIDB 在 Serverless 架构下提供弹性、高可靠的资源调度，使开发者能够快速部署、按需扩缩容，在复杂多变的 AI 工作负载中保持成本可控与性能稳定。

AI Database 是一款专为 AI 应用设计的全功能数据库，集成数据分析、多模态检索（向量/全文/JSON/GIS）与 ACID 事务于一体，可作为 AI 后端的统一数据服务平台。它兼容 PostgreSQL 协议，开发者可以用熟悉的 SQL 语法与工具直接接入，无需迁移即可获得更强大的 AI 原生能力。基于 Serverless 架构，AI Database 按需扩缩容，实现极致弹性与成本优化，帮助企业以更低成本、更高效率构建智能化应用与数据驱动的 AI 系统。

# 系统架构
AI Database 采用 存算分离 + 无状态计算节点池（Stateless Compute Pool） 架构，以对象存储为中心构建高可靠、弹性伸缩的 Serverless 数据平台。通过将计算、缓存、存储和日志服务解耦，系统在性能、成本与可维护性之间实现最优平衡。

## 核心能力

- **多模态检索一体化**​：向量检索支持高维相似度计算，适配大模型嵌入向量快速匹配；全文检索内置倒排索引，实现关键词高效查询与模糊匹配；JSON 检索原生支持文档存储与路径查询；GIS 空间检索兼容 PostGIS 接口，覆盖距离与包含等空间关系计算。

- **AI-Native 设计**​：ACID 事务保障训练数据与业务数据的强一致性，避免脏读与幻读；SQL 分析与实时计算能力让 BI 工具与 AI 模型在统一数据底座上协同。
        
- **Serverless 架构**​：●秒级启停，负载驱动扩缩容，冷启动时间小于 1 秒，适配推理服务、批处理等间歇性访问场景。
        
        

## 产品优势
    
- **数据可靠性**​：采用存算分离架构，计算节点无状态，数据持久化到对象存储 US3，天然具备多副本与高可用能力。
- **协议兼容**：100% 支持 PostgreSQL 协议，开箱即可接入现有生态工具与驱动，兼容版本覆盖 PostgreSQL 17.3。
- **高性能**：通过内核参数调优与针对分析、向量检索的优化路径，在复杂 SQL 与语义检索场景下提供极致性能。
- **快速部署**：支持秒级实例创建、扩缩容与配置调整，满足快速迭代与突发业务需求。
- **更低成本**：即时开通、按需付费，并支持 Auto Suspend 自动启停能力，根据业务负载动态控制资源占用。

## 典型场景
- **智能客服与知识问答**：整合企业知识库、FAQ 与工单数据，通过向量与全文双引擎实现语义检索与精准匹配，为大模型提供实时、可信的回答依据。
- **个性化推荐与内容检索**：存储商品、内容和用户行为的多模态向量特征，配合事务能力保障实时写入更新，提供低延迟推荐与混合搜索体验。
- **研发数据中台** ：集中管理日志、实验记录、模型输出等异构数据，依托 SQL 分析与 JSON/GIS 查询支持模型训练、评估与可视化。
- **AI Agent 任务执行**：通过 PostgreSQL 兼容协议让 Agent 直接调用企业数据，借助 Serverless 弹性应对任务高峰，确保 Agent 系统在不同负载下稳定运行。
