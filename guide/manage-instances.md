# 实例管理

本节介绍如何在 AI Database 中创建、查看和删除实例，帮助您完成实例的全生命周期管理。


## 创建实例

1. 登录 UCloud 控制台，进入“智能数据分析 MAXIR“产品控制台。

2. 在“AI 数据库“页签下，点击”创建实例“。

    ![](/images/001.webp)

3. 在“创建实例”页面，按需完成配置。

    ![](/images/002.webp)

4. 确认实例配置信息与付费信息，点击“立即购买”。


## 查看实例详情

1. 在“AI 数据库”页签下，找到对应的实例。

2. 点击实例“操作”列中的“详情”按钮。

    ![](/images/003.webp)

在该详情页中，你可以查看如下信息：

- “概览”页签：实例的基本信息，包括：实例名称、ID、接入地址等。

- “监控信息”页签：实例的监控信息，包括：用户总表数、正在运行的查询数、每秒写入的记录数等。

- “用户管理”页签：当前的数据库账号列表，包含每个账号的名称、创建时间。

- “操作日志”页签：实例的操作日志记录，包括每一个操作事件的操作时间、类型、操作内容、源 IP 地址、操作人等。


## 删除实例

1. 在“AI 数据库”页签下，找到对应的实例。

2. 点击实例“操作”列中的“删除”按钮。

3. 在弹出的对话框中，确认删除信息无误，点击“确认”按钮。

    ![](/images/004.webp)
